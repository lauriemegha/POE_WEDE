# BADDIES2BADDIES Website

## Project Overview

BADDIES2BADDIES is a luxury hairstyling brand based in Cape Town that specialises in delivering high-quality and customised hairstyles. The brand focuses on combining creativity, precision, and modern hairstyling trends to provide clients with a professional and unique experience.

The purpose of this website is to strengthen the online presence of BADDIES2BADDIES by showcasing the services offered by the brand and providing users with an accessible and user-friendly platform to explore hairstyles and request appointments. The website was designed with a modern and visually appealing interface that reflects the luxury identity of the brand while maintaining clear navigation and usability.

The website consists of multiple interconnected pages, including a homepage, services page, about page, contact page, and booking page. The homepage introduces the brand and highlights the main services offered. The services page presents different hairstyle categories, supported by images and pricing information, allowing users to compare options and make informed decisions.

The about page provides background information about the hairstylist and the vision behind the brand, while the contact page includes contact details, social media information, a contact form, and a map location to improve communication and accessibility. The booking page enables users to select a service, choose hairstyle preferences, and submit appointment requests through a structured booking form.

Overall, the website aims to improve client engagement, enhance the visibility of the brand, professionally showcase hairstyling services, and create a more convenient booking experience for users.
